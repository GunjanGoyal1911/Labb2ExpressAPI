Product API Documentation

GET /products

To get all products.

GET /products
        [
            {
                "id": 1,
                "name": "Apple",
                "type": "Fruit"
            },
            {
                "id": 2,
                "name": "Orange",
                "type": "Fruit"
            }
            //more products..
        ]

GET /product/:id

To get a specific product by ID.

GET /product/:id
        {
            "id": 1,
            "name": "Apple",
            "type": "Fruit"
        }

POST /product

Add a new product.

POST /product

Request Body Example:

{
        "name": "Product Name",
        "type": "Product Type"
    }

PUT /product/:id

Update a product by ID.

PUT /product/:id

Request Body Example:

{
        "name": "Updated Product Name",
        "type": "Updated Product Type"
    }

DELETE /product/:id

Delete a product by ID.

DELETE /product/:id

Notes:

  • All endpoints communicate with the external API at https://gunjanproductapp.azurewebsites.net/.
  • Replace {id} in the URLs with the actual product ID.
